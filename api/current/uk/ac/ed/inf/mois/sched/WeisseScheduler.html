  7. def apply(t: Double, tau: Double, group: ProcessGroup): (Double, Double)

    A single iteration of the scheduler must return the time after the iteration, and a suggested time step for the next iteration.

    A single iteration of the scheduler must return the time after the iteration, and a suggested time step for the next iteration. The parameter tau is treated as advice and the scheduler may run for a shorter time in order to be able to adapt the step size to control errors.

    t

    the starting time for the step

    tau

    the (suggested) size of the step

    group

    the process group to run

    returns

    time at end of step and new step size
