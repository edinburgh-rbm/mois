StochasticReactionNetwork

abstract class StochasticReactionNetwork extends BaseProcess with PopulationBasedReactionNetwork with KineticCatalyticReactionNetwork

Base trait for all reaction networks that use concentrations of molecules as a measure for species (as opposed to population-based reaction networks).
